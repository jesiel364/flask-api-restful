# flask-api-restful
API restful made with Python and Flask

## GET
Get all users <br>
`https://flask-api-restful.vercel.app/users/all`

## POST
Add a user <br>
`https://flask-api-restful.vercel.app/users/` <br>
`{
    "name": "Jason Vercel", "created": "06/07/2023"
}`
<br><br> "created" is no required

## DELETE
Remove a user <br>
`https://flask-api-restful.vercel.app/users/` <br>
`{
    "name": "Jason Vercel"
}`
